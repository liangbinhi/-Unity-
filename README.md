# Unity Grid Building System

一个基于 Unity 实现的网格建造系统，支持建筑旋转、网格吸附、占格检测与放置反馈，适用于建造、生存、RTS、模拟经营等类型游戏。

## 项目简介

本项目实现了一个完整的网格化建造系统。玩家可以在固定网格中放置不同尺寸的建筑，系统会根据建筑大小和朝向自动计算占用区域，并检测当前位置是否允许放置。

项目主要实现了以下功能：

- 网格化建筑放置
- 建筑旋转
- 多格子建筑占用计算
- 放置冲突检测
- 世界坐标与网格坐标转换
- 放置后的事件反馈与音效触发

该系统适合作为以下类型项目的基础模块：

- 城市建造类游戏
- RTS 即时战略类游戏
- 生存建造类游戏
- 模拟经营类游戏
